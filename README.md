# NETFLIX-CLONE-HTML-CSS

🎬 Netflix Clone (HTML & CSS)

This is a simple Netflix homepage clone built using HTML and CSS.
The project is created for learning and practicing frontend development skills.

---

🚀 Features

- Netflix-like homepage UI
- Hero section with background image
- Movie/show thumbnails layout
- Clean and modern design

---

🛠 Technologies Used

- HTML5
- CSS3

---

📂 Project Structure

- index.html
- style.css
- images/

---

▶️ How to Run

1. Download or clone this repository
2. Open "index.html" in your browser
